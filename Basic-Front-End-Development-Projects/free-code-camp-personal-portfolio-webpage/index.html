Writing

The Ruby Pupil Blog is a platform for me to learn and document all of the meaningful words of wisdom, revelations, teachable moments, cautionary tales, and successes that I gather during my journey from the Graphic Design Industry into Web Development/Computer Programming. There is a tremendous amount of knowledge that an individual must understand to be effective in the web industry, and my belief is that the transition will be much smoother if there are more resources that help explain web development from a beginner's perspective. The following blog consists of answers to the looming questions that go unasked and unanswered; all of which come from my beginner's perspective.
